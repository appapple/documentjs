documentjs.configured.documentConfig  
  • source

function(docConfig, project, parent, options, parentProject)

Parameters

  1. docConfig {Object}

    A docConfig loaded from {project.path}/documentjs.json.

  2. project {Object}

    A projectConfig object with data about the project being documented.

    • path {String}

      The location of the project being documented.

  3. parent {*}

  4. options {Object}

    command line overwrites.

  5. parentProject {Object}

    A projectConfig object with data about the most parent project being documented.

    • path {String}

      The location of the most parent project being documented.

    • The {docConfig}

      docConfig loaded from that project.
